🔑 Direct Posting Cookie Setup Guide

Direct headless publishing runs locally and requires browser session cookies in JSON format:

  2. Log in to Threads, Instagram, or Facebook.

  4. Save under the agent folder as config/cookies_<platform>.json (e.g. cookies_instagram.json).

🔑 Google Analytics 4 API Setup Required

To replace the dummy statistics and load real-time click and conversion reporting from Google Analytics 4 (GA4), please configure your Google Cloud credentials. Follow the steps below:

  1. Go to the Google Cloud Console and create a new project.
  2. Enable the Google Analytics Data API for your project.
  3. Create a Service Account and generate a new key in JSON format.
  4. Save this JSON key file in your workspace config directory as config/credentials-ga.json.
  5. Go to your Google Analytics 4 Admin Panel, select Property Access Management, and add the Service Account's email address (e.g. my-service-account@...iam.gserviceaccount.com) with Viewer access.
  6. Find your GA4 Property ID in the Property Settings, and add it to your .env file: GA4_PROPERTY_ID=123456789.
  7. Restart your local server and refresh this page.
